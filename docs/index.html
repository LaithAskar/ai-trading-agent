What's in the box

  • Backtest engine with a pinned lookahead-safety contract (orders at bar t fill at bar t+1's open), configurable slippage and commission.
  • ReAct agent driven by Claude with eight tools — list strategies, read strategy code, run backtest, search memory, get run details, compare runs, list SEC filings, fetch filing text.
  • Structured SQLite memory indexed by strategy / symbol / Sharpe — past runs are skippable on the next session.
  • Per-session cost cap hard-kills the loop if the dollar or token budget is exceeded.
  • Replay command re-runs an old session's tool calls against current code, surfaces semantic drift (timestamps and run IDs redacted).
  • Quant rigor pack: every backtest reports the buy-and-hold benchmark + Sharpe t-stat / p-value; walk-forward and param-sweep CLIs for honest validation.
  • SEC EDGAR data source via edgartools, including a filings-sentiment strategy with a Claude-driven variant for LLM-based signal extraction.
  • Alpaca paper-trading runner on the same Strategy interface. Constructor refuses live mode by design (raises NotImplementedError).
  • Remote MCP client on the official mcp SDK — connect to any OAuth-protected MCP server and merge its tools into the agent's registry.
